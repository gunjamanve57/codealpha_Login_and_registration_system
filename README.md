# Login and Registration System (C++)

## Description

This is a simple Login and Registration System developed in C++ using Object-Oriented Programming (OOP) and File Handling. It allows users to register with a username and password, and then log in using their credentials.

## Features

* User Registration
* User Login
* Duplicate Username Check
* Password Validation (Minimum 6 Characters)
* File Handling using `users.txt`
* Maximum 3 Login Attempts

## Technologies Used

* C++
* OOP (Classes)
* File Handling (`ifstream` and `ofstream`)

## How to Run

1. Compile the program:

   ```bash
   g++ main.cpp -o login
   ```

2. Run the program:

   ```bash
   ./login
   ```

## Project Files

* `main.cpp` - Main source code
* `users.txt` - Stores registered usernames and passwords
